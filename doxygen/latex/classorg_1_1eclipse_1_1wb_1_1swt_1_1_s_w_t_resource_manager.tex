\doxysubsection{Detailed Description}
Utility class for managing OS resources associated with SWT controls such as colors, fonts, images, etc. 

!!! IMPORTANT !!! Application code must explicitly invoke the {\ttfamily \mbox{\hyperlink{classorg_1_1eclipse_1_1wb_1_1swt_1_1_s_w_t_resource_manager_ae2266672ae9c04dd76064692cda459b5}{dispose()}}} method to release the operating system resources managed by cached objects when those objects and OS resources are no longer needed (e.\+g. on application shutdown) 

This class may be freely distributed as part of any application or plugin.
